# retail_sales_analysis
Objective
The objective of this project is to analyze retail sales data and identify business trends, revenue patterns, and product performance.
Dataset Description.

The dataset contains:

- Product Name
- Product Price
- Quantity Sold
- Region
- Revenue

Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

Analysis Performed

1. Data Inspection
2. Missing Value Analysis
3. Duplicate Record Analysis
4. Revenue Calculation
5. Product-wise Revenue Analysis
6. Region-wise Revenue Analysis
7. Sales Distribution Analysis
8. Price vs Revenue Analysis
9. Best Selling Product Identification

Findings

- No missing values were found.
- No duplicate records were detected.
- Phones recorded the highest sales quantity.
- Revenue differs across regions.
- Product pricing directly influences revenue.
- Retail sales data helps identify profitable products and regions.

Conclusion

The analysis demonstrates how retail businesses can use data to understand customer purchasing behavior, identify top-performing products, and improve revenue generation. Data-driven decisions can improve sales performance and business growth.

Outcome

This project successfully applies data science techniques to a real-world retail business scenario and generates meaningful insights from sales data.
